## TO-DO List Manager

### Commands to use me

* `add` : Adds a task to the list
* `delete` : Deletes the specified task
* `mark` : Marks the task as done
* `unmark` : Marks the task as incomplete
* `bye` : To switch me off!

### Example usage 

* `add` : `add read book`
* `delete` : `delete 1`
* `mark` : `mark 1`
* `unmark` : `unmark 2`
* `bye` : `bye`
